
--> Spanning Tree : A tree in which we have N nodes and N - 1 edges and all nodes are reachable from each other !!!

--> We can generate more than one Spanning Tree for a given connected graph  !!! --> Tree with the smallest edge weight sum will be the MST.

--> For more understanding refer this video : https://www.youtube.com/watch?v=ZSPjZuZWCME&list=PLgUwDviBIf0oE3gA41TKO2H5bHpPd7fzn&index=46

--> No. of vertices in a Graph = No. of Vertices in MST !!!

--> How to find the MST for a given graph ??
